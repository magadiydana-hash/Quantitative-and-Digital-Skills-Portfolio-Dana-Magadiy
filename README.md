# Quantitative-and-Digital-Skills-Portfolio-Dana-Magadiy
Project Overview
This project involves a comprehensive audit of a transaction-based customer dataset (31,000+ records) spanning from 2021 to 2024. The goal was to segment the customer base, analyze profitability concentration, and evaluate retention trends using cohort analysis.

Digital & Quantitative Skills Demonstrated
Data Segmentation: Performed Decile Analysis to identify high-value customer segments (Top 10% generating ~40% of total profit).

Cohort Analysis: Developed a retention matrix to track revenue decay and acquisition performance across yearly cohorts (2021–2024).

Financial Metrics: Calculated key performance indicators, including:

AOV (Average Order Value)

AOF (Average Order Frequency)

Customer Equity and Profit Margin Analysis.

Advanced Excel: Utilized Pivot Tables, complex lookup functions (VLOOKUP/INDEX-MATCH), and conditional formatting for data visualization.

Repository Structure
Customer_Base_Audit_Analysis.xlsx: The core analytical engine with cleaned data and calculations.

Customer_Base_Synopsis.pdf: A one-page professional summary linking findings to economic theory.

Key Results
Identified that the top 2 deciles account for over 56% of total company revenue.

Observed a significant surge in the 2024 cohort, contributing 152.7M KZT in new revenue.

Calculated an average margin of 48% across all active customer segments.
